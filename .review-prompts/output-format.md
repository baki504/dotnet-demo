# レビュー出力フォーマット

レビュー結果は以下のフォーマットで出力してください。

## スコア

- **総合スコア**: X / 10（10点満点で評価）
- **指摘件数**: 🔴高 0件 / 🟡中 0件 / 🟢低 0件

### スコアリング基準

| スコア | 基準 |
|--------|------|
| 9-10 | 指摘なし、または低優先度の軽微な指摘のみ |
| 7-8 | 中優先度の指摘あり（改善推奨だが動作・保守に大きな支障なし） |
| 5-6 | 高優先度の指摘あり（品質・保守性に明確な問題） |
| 3-4 | 高優先度の指摘が複数あり（設計や安全性に重大な懸念） |
| 1-2 | 根本的な見直しが必要（目的を果たせていない） |

## 概要

レビュー対象の全体的な品質と主要な所見を2〜3文で要約。

## 指摘事項

各指摘を以下の形式で記載：

### [指摘タイトル]

- **優先度**: 🔴高 / 🟡中 / 🟢低
- **理由**: なぜこれが問題なのか
- **対処法**: 具体的な修正方法
- **効果**: 修正することで得られる改善

指摘がない場合は「指摘事項なし」と記載。

## 補足

レビュー観点には直接該当しないが、改善の余地がある点や参考情報があれば記載。なければ省略可。
